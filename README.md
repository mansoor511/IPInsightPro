# 🔍 IP Insight Pro - ابزار پیشرفته اطلاعات IP

## 📌 معرفی
IP Insight Pro یک ابزار حرفه‌ای و قدرتمند برای دریافت اطلاعات دقیق از آدرس‌های IP است. این ابزار مناسب برای مدیران شبکه، محققان امنیتی و هکرهای اخلاقی می‌باشد.

## ✨ قابلیت‌ها
- 🔍 دریافت اطلاعات IP خودتان
- 🎯 آنالیز IP هدف
- 🌍 موقعیت جغرافیایی روی نقشه
- 🔄 جستجوی معکوس DNS
- 📊 نمایش رکوردهای DNS
- 📋 اطلاعات WHOIS
- 📡 محاسبه محدوده IP
- 🔢 بررسی چند IP همزمان
- 🗺️ لینک مستقیم Google Maps
